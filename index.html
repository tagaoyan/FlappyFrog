FlappyFrog (VMO1)

This project was forked from tusenpo / FlappyFrog, and was originally hosted at https://moxia.cf2. I appreciate the idea of this game and forked it. Then I did some little modifications on it.

Stories

I found the original repository because I played the game and wanted to get the sound files of it. I attempted to access the /sounds directory and found a 404 page of Github Pages. Then I searched Github with some distinctive identifiers used in the code and found it. I had not known this page on v2ex before.

Updates

I did three things on this project.

  1. Let the sprite die when hitting the ceiling. This is a common design for flappy games.
  2. Let the sprite change appearance when game is over. The ending of this game reminded me of a picture I had seen3.
  3. Change the ground image. This is essential for the consistency of all scenes.

Sorry for having done such little work, Thanks.

How to Play

To apply for player, you should:

  1. Clone this repository to your computer. Use git. Downloading an archive is allowed but not welcome.
  2. Run a web server with the path of the repository as root path. For example, you can run python3 -m http.server in the path you just cloned. In Python 2.x this package has a different name containing Simple. If you want to follow the progress of history and avoid being too simple, use Python 3.x.
  3. Open your web browser and visit the home page of the local web server. If you followed above steps, it will be http://localhost:8000/.
